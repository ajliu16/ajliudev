Poequery - Web Project
Poequery.com, Source Code
PoeQuery is a website designed to search and filter in-game Unique items in the popular ARPG Path of Exile. It is written in Python, HTML, CSS, and Javascript.
Back End / Scripting
Python is used to parse through a list of item information and instantiate them as Unique objects.
Attributes such as DPS or Armour are calculated, and the objects are appended into a list.
Another class takes in this list, containing methods to create JSON files of each filtered catagory (i.e. Axes with the highest DPS).
The Intersection Observer API is used to lazy load item images.
Front End
The front end is written in plain HTML, CSS, and Javascript, with the exeption of Ken Wheeler's slick slider implemented with jQuery.
By clicking a catagory, the JSON file with the matching catagory is fetched and used to update the DOM.
The search bar is implemented in Javascript, searching for objects with matching text. A slight delay after key inputs is added to improve search performance.
